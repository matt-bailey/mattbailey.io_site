Say Hello To My Shiny New Site

22 September, 2014

My old Tumblr blog served me well, but I’ve been wanting a bit more control for a long time. Say hello to my shiny new, Jekyll powered site!

It’s nice to finally have complete control over the code and style, and it’s ‘self-hosted’ by those wonderful folks over at Site5. It’s still a work-in-progress, so expect to see things change from time-to-time.

My main aim was to simply create a nice reading environment, which I think I’ve achieved - I particularly love Jubilat as a body font.

If you want to find out more about the tech specs of the site check out the About Me page. If you have any comments, questions or feedback use the form below.
